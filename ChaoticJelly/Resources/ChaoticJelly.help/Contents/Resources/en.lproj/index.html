Chaotic Jelly Help

Chaotic Jelly scans your video library, identifies non-English subtitle and audio tracks, and removes them — saving disk space and keeping your media clean for Jellyfin.

Getting Started

  1. New Scan — Select a folder containing video files. Chaotic Jelly scans recursively for MKV, MP4, M4V, AVI, MOV, TS, and M2TS files.
  2. Choose a mode — Remove subtitles only, audio only, both, or optimize for Jellyfin direct play.
  3. Review — Inspect every file and planned action before processing. Use dry-run to preview without changes.
  4. Process — Files are copied locally, processed, validated, and then the original is replaced.

Processing Modes

  • Remove non-English subtitles only — Keeps all audio, removes subtitle tracks that aren't in your keep-language list.
  • Remove non-English audio only — Keeps all subtitles, removes audio tracks that aren't in your keep-language list.
  • Remove both — Removes both non-English subtitles and audio tracks.
  • Optimize for Jellyfin — Additionally remuxes or transcodes for Jellyfin direct play compatibility.

Language Handling

Chaotic Jelly uses language tags embedded in video files (ISO 639 codes like "eng", "fre", "ger") to determine which tracks to keep or remove.

Conservative Mode

When enabled (default), tracks with missing or ambiguous language tags are kept rather than removed. This prevents accidental data loss.

Preserved Tracks

  • Forced subtitles — Always preserved (e.g., translations of foreign-language dialogue within an English film).
  • SDH subtitles — Optionally preserved (hearing-impaired subtitles).
  • Commentary tracks — Optionally preserved.

Network Share Safety

Files on network shares (SMB/NFS) are never processed in place. Chaotic Jelly:

  1. Copies the file to a local cache
  2. Processes the local copy
  3. Validates the output (duration, streams, integrity)
  4. Atomically replaces the original only after validation passes

Settings

Language

Configure which languages to keep. Default is English ("eng"). Add additional language codes (e.g., "spa" for Spanish, "fre" for French).

Tools

Chaotic Jelly uses FFmpeg, FFprobe, and optionally MKVToolNix. Configure custom paths or use the bundled/auto-detected binaries.

Cache

Set the local cache path and maximum size. Cache is used for safe network share processing.

Keyboard Shortcuts

  • ⌘1–⌘6 — Switch between Dashboard, Scan, Review, Queue, History, and Logs
  • ⌘, — Open Settings
  • ⌘R — Refresh current view

Troubleshooting

"Tool not found" error

Install FFmpeg via Homebrew: brew install ffmpeg. Or configure custom tool paths in Settings → Tools.

Processing fails on a file

Check the Logs tab for detailed error output including the exact FFmpeg command and exit code. Failed files can be retried from the History tab.

File on network share is locked

Ensure no other application (Jellyfin, Plex) has the file open. Chaotic Jelly will retry once, then skip locked files.
